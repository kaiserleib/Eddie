# Sample Songs

Want to contribute a song to Eddie? We'd love to include your composition!

## How to Submit a Song

1. Compose your song in Eddie
2. Copy the hash from the URL (everything after `#`)
3. Fork this repo and add your song to `songs.json`
4. Open a PR with your changes

## songs.json Format

Add a new entry to the array:

```json
{
  "name": "Your Song Name",
  "hash": "your-hash-here"
}
```

Note: URLs are now LZ-compressed for shorter sharing. Old uncompressed URLs still work.

## Guidelines

- Original compositions or public domain works only
- Test playback before submitting
- Have fun!
